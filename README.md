MExBtcAPI
=========


Project Goals
=============

* provide a set of well developed classes useful in representing concepts like  
    * currency
    * money
    * exchange rate
    * currency exchange
    * exchange order
* promote their use
* provide API implementations of various bitcoin exchanges using them
